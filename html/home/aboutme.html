关于"我"

国内外技术社区很多，各自都有一些优秀的文章 和 代码资源
在这里收集和汇总一些内容，信息都是来自于各个不同的社区

在这里，你可以找到：

一系列文章
汇总一些新发布的，或者已发布旧的系列型的技术文章，方便查看和回顾

一小段代码
通过运行这段代码，了解程序的运行
也可以作为您工作中，能够解决小问题的一种便捷方式

收集的内容

1. CSS代码片段
2. 后台代码片段（部分支持语言）
3. 系列文章收集（包含前端、后端等不同技术领域的系列教程）
4. 开发站点资源：技术官网、开发社区、WIKI
5. 面向开发的API收集（例如七牛、青云等）

关于"站长"

后端经历：

名不见经传的中年码农，正在前端这条路上摸爬滚打

我从较旧的ASP + ACCESS开始折腾
然后经历了ASP.NET + ORACLE / MSSQL洗礼之后

后来又投靠了PHP + MySQL，并游历与不同浏览器之间的兼容性问题（IE7 - IE8，firefox，safari，chrome）
期间主要从事网站和信息系统开发，接触过Joomla、 Wordpress、Codeigniter、 Thinkphp等开发框架

机缘巧合之下，接手APP接口开发工作，就这样接触了java后台开发，Spring MVC + Mybatis
大部分是基础的业务逻辑功能开发，即数据库的增删改查
偶尔会接触到线程、Spring quartz定时器等

前端经历：

专职的前端工作始于2016年，一开始接触的是AMD、CMD开发模式，主要开发工作是基于sea.js + jquery
随着Node.js技术栈的发展，前端的开发工作慢慢的转向Vue.js技术栈，主要基于组件式开发
同时，也逐渐接触和实践小程序开发

关于未来：

了解更多与前端相关的技术原理，比如：垃圾回收，浏览器原理，JS引擎工作原理等
接触不同的技术：机器学习、区块链开放、物联网等
深入学习计算机基础：数据结构、设计模式、计算机体系结构等
